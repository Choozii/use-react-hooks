# 훅 이름

## Introduce

훅에 대한 설명
(모든 설명은 '-다.'로 끝맺기)

```ts
// 훅 인터페이스
interface UseSomethingHookProps {}

interface UseSomethingHookReturns {}

const useSomethingHook = (props: UseSomethingHookProps): UseSomethingHookReturns
```

- 훅 인터페이스에 대한 상세한 설명
- 또는 부가 설명이 필요한 것에 대한 내용
